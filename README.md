# Comment System APP

This app demonstrates how to develop a comment system APP by using with React.js, Redux & Sass

## Features

- Authorization
- Add, edit, delete comments
- Like, dislike comments
- Filter & Paginate comments

## Installation

```bash
$ npm install
```

## Running app
```bash
# development
$ npm run dev

# production mode
$ npm run build
$ npm run preview
```
